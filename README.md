# shell-cpp

A simple mini-shell written in C++.

## Features

* Execute basic shell commands
* Built-in command handling
* Basic flag support
* File system operations: `cd`, `pwd`, `touch`

## Build

```bash
make
```

## Run

```bash
make run 
```
or 

```bash
./target/main
```

## Example

```bash
touch file.txt
cd /path/to/directory
pwd
```
